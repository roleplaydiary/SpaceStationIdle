Из таблицы "Отделы улучшения" выгружаем данные с помощью плагина. Переименовываем файл в "upgrade_data_google.json", кладём в папку Data в проекте, где лежит скрипт "sync_upgrades_from_json.py".
Через командную строку запускаем скрипт. Потом открываем проект, выбираем SO с апгрейдами, удаляем старые записи и импортируем обновленный файл.
